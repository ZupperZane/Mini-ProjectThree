# Mini-ProjectThree #
### By: Hayden Jones

A Grocery App to help organize a total, with an approximate total.
## Component-based implementation & at Least one usage of Reduce
Cards.jsx is a separate file using imports to separate the Cards and Products into a different file. In Cards holds a function getTotal, using reduce on the curCart to calculate the total with 7% tax.
## Cart
### Add/Insert ,Delete/Remove, Update Existing Item
The Cart component comes with 3 buttons and 2 increments satisfying add, delete, and update, as cart items and total update alongside the count of individual items.
### Display the current list of items
The Cart displays items, count, clearing of cart, and approximate tax
